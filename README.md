# ScopeShield

> Write proposals in 2 minutes. Lock scope forever.

ScopeShield generates a professional Proposal + Statement of Work + Change Request Email for freelancers using a structured Scope Builder form + AI polishing. Downloads as an editable DOCX.

**Not legal advice.** All generated documents are for business communication purposes only.

---

## Fresh clone → running locally in under 20 minutes

### Prerequisites
- Node.js 18+
- npm 9+
- [Stripe CLI](https://stripe.com/docs/stripe-cli) (for webhook testing)

### 1. Install
```bash
npm install
cp .env.example .env.local
```

### 2. Configure environment variables
Open `.env.local` and fill in all values. You need accounts at:
- [Clerk](https://clerk.com) — free plan works
- [Neon](https://neon.tech) — free plan works
- [Anthropic](https://console.anthropic.com) — pay-as-you-go
- [Stripe](https://stripe.com) — test mode

See [docs/runbook.md](docs/runbook.md) for step-by-step instructions for each service.

### 3. Run database migration
```bash
npx prisma migrate dev --name init
npx prisma generate
```

### 4. Start the dev server
```bash
npm run dev
# → http://localhost:3000
```

### 5. Run tests
```bash
npm test           # Jest unit tests
npm run typecheck  # TypeScript check
```

## Stripe setup (required for billing)

In your Stripe dashboard, create two products:
1. **ScopeShield Pro Monthly** — $14/month recurring subscription
2. **ScopeShield Pro Annual** — $119/year recurring subscription

Copy the price IDs to your `.env.local`:
```
STRIPE_PRICE_PRO_MONTHLY=price_...
STRIPE_PRICE_PRO_ANNUAL=price_...
```

### Test webhooks locally
```bash
stripe listen --forward-to localhost:3000/api/webhooks/stripe
```

Use the webhook signing secret printed by the CLI as your `STRIPE_WEBHOOK_SECRET`.

## Architecture

| Layer | Tech |
|---|---|
| Frontend | Next.js 14 (App Router) + TypeScript + Tailwind |
| Auth | Clerk v5 (`User.id = Clerk userId`) |
| Database | Neon Postgres + Prisma |
| AI | Anthropic Claude Haiku |
| Payments | Stripe subscriptions |
| DOCX | `docx` npm package |
| Hosting | Vercel |
